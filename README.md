# 🍰 Vkusnyashka City — интернет-магазин кондитерской

## 📌 Бизнес-задача
> Автоматизировал прием заказов для кондитерской в Астане, внедрив отправку корзины в WhatsApp, систему промокодов и админ-панель для управления меню.

## 🚀 Демо
[**👉 Перейти на сайт**](https://aligalymuli-jpg.github.io/vkusnyashka_city/index.html)  


## 🛠 Технологии
- **Frontend**: HTML5, CSS3 (Glassmorphism, адаптив), JavaScript (ES6+)
- **Backend & Cloud**: Firebase Realtime Database, Firebase Auth, ImgBB API
- **Интеграции**: WhatsApp (wa.me), localStorage для корзины
- **Деплой**: GitHub Pages / любой хостинг статики

## 📋 Что сделано
- ✅ Каталог товаров с фильтрацией по категориям (торты, пироги, выпечка, напитки, сеты)
- ✅ Корзина с сохранением в localStorage, изменение количества, расчет суммы
- ✅ Умная система промокодов (одноразовые / многоразовые, % или фикс)
- ✅ Интеграция с WhatsApp: автоматическое формирование заказа с адресом, телефоном, приборами
- ✅ Прогресс до бесплатной доставки (от 15 000 ₸)
- ✅ Админ-панель: CRUD товаров, загрузка фото на ImgBB, управление промокодами
- ✅ Страница кейтеринга для банкетов и выездного обслуживания
- ✅ Адаптивный дизайн (Mobile First), фоновое видео, прелоадер

## 📈 Результаты для бизнеса
- Автоматизация заказов → экономия времени персонала ~30-40%
- Механика бесплатной доставки увеличила средний чек (клиенты добирают товары до 15 000 ₸)
- Система промокодов стимулирует повторные продажи и привлекает новых клиентов
- Админ-панель позволяет менять цены и акции без программиста
